## Project Description
Project MindMirror is designed to **assess students with puzzle like questions** that challenge one's ability in each type of intelligence. After the quiz, the user's score is displayed, helping them **identify their strengths** in different types of intelligence.

## Features
-Display puzzles that tackles different topics\
-Calculate score in the end\
--Utilizes a polar bar chart to display how well you perform in a type of intelligence
