# Project Title: MindMirror

## Project Description
Our python program will consist of puzzles to test students. In the end of the puzzles a score will be calculated to show how much you scored in different forms of intelligence.

## Features
-Display puzzles that tackles different topics 
----Calculate score in the end
----Utilizes a polar bar chart to display how well you perform in a type of intelligence

## How to Run the Program
-Have python installed
----Download the file: 'MindMirror'
