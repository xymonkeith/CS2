## Project Description
Students often compare test scores among each other, creating tension, and making others feel worse. MindMirror solves that problem by letting users take a **puzzle-based quiz that challenges their ability in various types of intelligences.** At the end of the session, a chart is displayed reflecting their actual performance, showing students that academic scores do not display intelligence, as there are other forms that are simply undiscovered.

## Features
-Display puzzles that tackles different topics\
-Calculate score in the end\
-Polar bar chart to display how well you perform in a type of intelligence
